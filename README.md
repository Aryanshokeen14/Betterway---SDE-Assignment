Mini E-Commerce Website

This project is a mini e-commerce website developed using React.js as part of a college assignment.
The objective of the project is to understand and implement basic frontend concepts such as product listing, cart management, filtering, and state handling.

PROJECT DESCRIPTION

The website displays a list of products fetched from a public API.
Users can browse products, search by name, filter by category, sort by price, and add products to a cart while respecting available stock.
The project focuses more on logic and functionality rather than UI design.




FEATURES

1.Product listing in grid layout

2.Products fetched using a public API

3.Search products by name

4.Filter products by category

5.Sort products by price

6.Add products to cart

7.Update product quantity

8.Prevent adding products beyond available stock

9.Show stock status (In Stock / Out of Stock)

10.Disable Add to Cart button when stock is exhausted

11.Cart total price calculation





TECHNOLOGIES USED

1.React.js

2.JavaScript(ES6)

3.HTML&CSS

4.FETCH API



API used - https://fakestoreapi.com/products




How to run the project

1.Install dependencies: npm install

2.Start the project: npm start

The application runs on http://localhost:3000.
